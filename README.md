Trading Sessions

A web-based trading sessions tracker designed for forex traders. This application displays major global trading sessions in real-time according to the user's local timezone, helping traders identify the best trading opportunities and market overlaps.

Features

- Display major forex trading sessions:
  
  - Sydney Session
  - Tokyo Session
  - London Session
  - New York Session

- Real-time session status
  
  - Active Session Indicator
  - Upcoming Session Information
  - Session Open and Close Times

- Session Overlap Detection
  
  - London–New York Overlap
  - Tokyo–London Transition
  - Other Market Intersections

- Countdown Timer
  
  - Time remaining until session opens
  - Time remaining until session closes

- Responsive Design
  
  - Mobile-friendly interface
  - Desktop compatibility
  - Modern and clean UI

Purpose

The purpose of this project is to help forex traders easily monitor global market sessions and identify high-liquidity trading periods without manually converting time zones.

Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design

Installation

1. Clone the repository:

git clone https://github.com/iali-asad/trading-sessions.git

2. Open the project folder.

3. Run the application by opening "index.html" in your browser.

Usage

1. Open the application.
2. View current active trading sessions.
3. Check session opening and closing times.
4. Monitor overlap periods for increased market activity.
5. Use countdown timers to prepare for upcoming sessions.
